About Seraphim

Security Services Overview

Layanan pengujian keamanan yang berfokus pada identifikasi risiko nyata dan solusi yang dapat langsung diterapkan.

Hubungi

Seraphim Security menyediakan layanan pengujian dan analisis keamanan untuk membantu mengidentifikasi serta mengurangi risiko pada sistem dan aplikasi.

Pendekatan yang digunakan berfokus pada praktik umum seperti OWASP dan PTES, dengan pengujian langsung terhadap potensi celah pada web, network, dan API.

Penetration Testing

Pengujian keamanan pada web, network, dan API.

Vulnerability Assessment

Identifikasi kerentanan dan rekomendasi perbaikan.

OSINT Investigation

Analisis informasi terbuka untuk identifikasi risiko.

Security Audit

Pemeriksaan sistem dan konfigurasi keamanan.

API Security

Pengujian keamanan API REST dan GraphQL.

Incident Response

Analisis kejadian keamanan dan penanganannya.
